# Django_project
 Проект магазина реализованный на фраймвёрке Джанго. Состоит из двух блоков - блок с пользователями и блок с товарами и заказами. Пользователь может зарегистрироваться, имеет свой собственный профиль и может иметь разный уровень доступа от покупателя до администратора. Товары можно добавлять/удалять/изменять/архивировать

Для запуска и проверки скачайте все файлы проекта на свой локальный репозиторий, установите все необходимые зависимости
выполнив команду - "pip install -> requirements.txt" находясь в корневой папке проекта.
Затем выполните в терминале команду - "python manage.py runsever"

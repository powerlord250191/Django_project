# Django_project
 Проект магазина реализованный на фраймвёрке Джанго. Состоит из двух блоков - блок с пользователями иблок с товарами и заказами. Пользователь может зарегистрироваться, имеет свой собственный профиль и может иметь разный уровень доступа от покупателя до администратора. Товары можно добавлять/удалять/изменять/архивировать
